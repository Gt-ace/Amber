This is a personal site running on [Amber](https://github.com/Gt-ace/Amber).

A few notes, occasional writing, the things I want online without
giving them to a platform.

## The setup

The whole site is markdown files in a git repo.

This is a project i'm building in the student project house in St. Gallen. Ideation and architectural decisions are done by me and claude, claude code does the implementation.
